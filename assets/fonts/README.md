# Mona Sans Font

Aplikasi memakai **Mona Sans** sebagai font utama. Empat file .ttf berikut **wajib** ada di folder ini agar app bisa jalan:

- `MonaSans-Regular.ttf`
- `MonaSans-Medium.ttf`
- `MonaSans-SemiBold.ttf`
- `MonaSans-Bold.ttf`

## Cara menambahkan

1. Download Mona Sans (TTF) dari https://github.com/github/mona-sans atau sumber resmi lain.
2. Salin/rename ke 4 nama file di atas lalu letakkan di folder ini.

Nama file harus persis seperti di atas.
